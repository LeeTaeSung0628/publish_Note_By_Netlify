C언어로 구현된 프로그램을 분석하고, 실행결과를 쓰시오.

#include <stdio.h>

int len(char* p);
int main( ) { 
	char* p1 = "2022"; 
	char* p2 = "202207"; 
	int a = len(p1); 
	int b = len(p2); 
	printf("%d", a + b);
} 

int len(char* p) {
	int r = 0; 
	while (*p != '\0') { 
		p++;
		r++; 
	} 
	return r;
}
  1. int len(char* p); -> len() 함수의 프로토타입 선언이다.
  2. p1과 p2는 문자형 포인터 변수로 각 문자가 저장된 주소의 주소로 초기화 된다.

    ex) p1에는 '2'가 저장되어있는 1000번째 주소가 삽입되어있으며,
    p1+1 은 '0' , p1+4 는 1004번째 주소로, '\0'이 들어가 있다.

즉, 4 + 6

C언어로 구현된 프로그램을 분석하고, 실행결과를 쓰시오.

#include <stdio.h> 
int main( ) { 
	int a[4] = { 0, 2, 4, 8 }; 
	int b[3]; 
	int* p; 
	int sum = 0; 
	for (int i = 1; i < 4; i++) { 
		p = a + i; 
		b[i - 1] = *p - a[i - 1]; 
		sum = sum + b[i - 1] + a[i];
	} 
	printf("%d", sum); 
}
  1. p -> 정수형 포인터 변수
  2. i = 1,2,3 반복문
첫번째 반복 i = 1 
	`p = a + i` => a배열의 두번째 요소의 주소 저장
	`b[0]` = (*p => p의 주소가 가리키는곳의 값)2 - 0 = 2
	sum = 0 + 2 + 2 = 4
두번째 반복 i = 2
	a배열의 세번째요소 4
	`b[1]` = 4 - 2 = 2
	sum = 4 + 2 + 4 = 10
세번째 반복 i = 3
	a배열의 네번째요소 8의 주소
	'b[2]' = 8 - 4 = 4
	sum = 10 + 4 + 8 = 22
  1. 포인터 변수로 선언된 p에 배열을 직접대입하는것은 주소값을 넣는것이다.
  2. 포인터 변수를 사용할때 p*처럼 사용하면, 해당 주소값 내의 값을 의미한다.
    p에 직접 대입할 때 말고는 다 배열과 사칙연산이다.

C언어로 구현된 프로그램을 분석하고, 실행결과를 쓰시오.

#include <stdio.h> 
int main() { 
	int* array[3]; 
	int a = 12, b = 24, c = 36; 
	array[0] = &a; 
	array[1] = &b; 
	array[2] = &c; 
	printf("%d", *array[1] + **array + 1);
}
  1. 정수형 포인터를 저장하는 배열 array [ , , ]
  2. array[0] = &a;는 a의 시작 주소를 삽입하는 것이다.
  3. array[1] 은 &b(b의 주소값)이고, *array[1] 은 그 주소가 지니고 있는 실제 값 24이다.
  4. **array + 1 =>
    • array : 배열의 이름만 지정하면 배열의 첫 번째 요소의 주소인 &array[0], 즉 array의 첫주소값
    • *array : array의 첫주소값(array)이 가지는 실제 값 &a 즉 int a의 주소값.
    • **array : int a의 주소값의 실제 지니고 있는값 12이다.

즉, 24 + 12 + 1

C언어로 구현된 프로그램을 분석하고, 실행결과를 쓰시오.

#include <stdio.h> 
int main() { 
	int ary[3]; 
	int s = 0; 
	*(ary + 0) = 1; 
	ary[1] = *(ary + 0) + 2; 
	ary[2] = *ary + 3; 
	for (int i = 0; i < 3; i++) 
		s = s + ary[i]; 
	printf("%d", s); 
}
  1. *(ary + 0) = 1; -> arr의 첫 주소값의 실제값에 1 대입 / arr[1 , , ]
  2. ary[1] = *(ary + 0) + 2; -> arr[1]에 1+2 대입 / arr[1 , 3 , ]
  3. ary[2] = *ary + 3; -> arr[2]에 1+3 대입 / arr[1 , 3 , 4]

1+3+4 = 8

C언어로 구현된 프로그램을 분석하고, 실행결과를 쓰시오.

#include <stdio.h> 
main() { 
	char* p = "KOREA"; 
	printf("%s\n", p); 
	printf("%s\n", p + 3); 
	printf("%c\n", *p); 
	printf("%c\n", *(p + 3)); 
	printf("%c\n", *p + 2); 
}
  1. 문자열을 저장하는 경우 문자열의 끝을 의미하는 널 문자\0 가 추가로 저장되며,
    출력시 널 문자는 표시되 지 않는다.
  2. printf("%s", P) 의 %s는 널(\0) 문자가 나올 때까지 문자열을 출력한다.

[K , O , R , E , A , \0]

  1. P : 문자형 포인터 변수 p 의 시작주소값 -> KOREA
  2. P + 3 : p의 시작주소값의 3번째 뒤의 주소 -> E의 시작 주소값 -> EA
  3. *P : p 의 시작주소값의 실제 값 -> K
  4. *(P + 3) : P의 시작주소값의 3번째 뒤의 주소의 실제 값 -> E
  5. *P + 2 : P의 시작 주소의 실제값 K(ASCII 코드 75) + 2 => 77 => M
